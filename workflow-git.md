\# Workflow Git simple



1\. \*\*branch\*\*  

&#x20;  Créer une nouvelle branche pour travailler sur une fonctionnalité.



2\. \*\*commit\*\*  

&#x20;  Enregistrer les modifications avec un message clair.



3\. \*\*push\*\*  

&#x20;  Envoyer la branche vers le dépôt distant (GitHub).



4\. \*\*pull request\*\*  

&#x20;  Proposer la fusion des modifications sur la branche principale.



5\. \*\*merge\*\*  

&#x20;  Fusionner la pull request dans la branche principale (main).
